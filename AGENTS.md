# AGENTS.md

本仓库前端 UI 调整以园区 RuoYi 截图为主，`模块迁移/UI/bladex-vue3-list-page-ui-reuse-checklist.md` 为辅助清单。若清单与截图存在冲突，优先遵守截图。

## Saber3 园区 UI 硬性约束

- 顶部导航栏宽度 100%，高度 70px，背景色 `#1059c6`。
- 顶部导航承载 LOGO、一级菜单和右侧账户区；一级菜单横向展示。
- 左侧菜单栏宽度 260px，背景色 `#ffffff`，仅承载当前一级菜单下的二级菜单。
- 标签栏高度 50px，背景色 `#eaecf2`。
- 主体区域背景色 `#f4f4f6`。
- 主体内容区四周页边距 22px。
- 内容模块背景色 `#ffffff`，圆角 10px。
- 模块之间横向间距 16px，纵向间距 16px。
- 顶部导航、左侧菜单、标签栏、主体区域必须形成统一后台工作台布局，不允许页面自建第二套导航。

## 首页工作台约束

- 首页必须贴近截图的运营工作台风格，不做普通列表页。
- 首页首屏结构为：顶部指标卡、左主右辅工作区。
- 左侧主栏包含 Banner、招商闭环入口、政策通知、快捷入口。
- 右侧辅栏包含日历 / 日程与待办提醒。
- 日历 / 日程区块底部必须与招商闭环入口底部对齐；待办提醒顶部必须与政策通知、快捷入口顶部对齐。
- Banner 必须具备图片或视觉背景，不允许纯文字占位。
- 首页区块沿用白底、10px 圆角、轻边框或轻阴影。

## 列表页约束

- 标准列表页优先沿用 `basic-container`、`stat-cards`、`avue-crud`。
- 查询区、工具栏、表格、分页、空状态使用统一视觉风格。
- 状态统一用 `el-tag`，颜色遵守业务语义：正常绿色、待处理蓝色、预警橙色、风险红色、中性灰色。
- 不允许每个业务页面单独硬编码一套主色、边框色或背景色。
